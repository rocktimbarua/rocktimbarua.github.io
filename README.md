# rocktimbarua.github.io# Rocktim Barua | Compliance Officer & Business Analyst Portfolio

Welcome to my professional portfolio repository. This project is a responsive, animated web application engineered to showcase interactive business intelligence frameworks, data summarization models, and regulatory compliance workflows.

## 🚀 Live Demo
You can view the live, animated website here:(https://rocktimbarua.github.io) 

## 📊 Core Highlights Showcase
* **Interactive Dashboards:** Visualized tracking sheets mapping cross-functional HR/Payroll outlays, critical MFS field KPIs, and clearing/forwarding cost optimization frameworks.
* **Analytical Matrix:** Highlighting automated Excel and Power BI reporting models that cut report generation delays by 25% and improve decision-making velocity by 15%.
* **Regulatory Compliance Framework:** Operational insights covering robust KYC/AML procedures, internal audit readiness, and comprehensive risk mitigation tracking.

## 🛠️ Technology Stack
* **Frontend Structure:** Semantic HTML5
* **Styling & Motion:** Modern CSS3 (Custom Variables, Keyframe Animations, Hover Transitions, and Responsive Flexbox/Grid Layouts)
* **Iconography:** Font Awesome Integration
